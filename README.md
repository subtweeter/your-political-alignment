# your-political-alignment
A static client-side web application built with HTML, Tailwind CSS, and Chart.js to calculate ideological alignment within the Iranian political spectrum. It evaluates 20 policy indicators using a cross-weighted scoring matrix against four factions, normalizing the output into an interactive radar chart to visualize political coordinates.
